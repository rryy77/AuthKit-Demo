# AuthKit Demo

AuthKit is a React + Supabase authentication starter kit.

## Features

- Email login
- Email signup
- Password reset
- Google OAuth
- Protected routes
- Dashboard page
- Dark mode
- Responsive design
- TypeScript
- Vite
- Supabase Auth
